What is Context-Grounded Bible?

An open-source initiative to create extensive contextual data for every verse in the Bible. By providing AI systems with comprehensive commentary (essentially "a book's worth of information" for each verse), we ground AI responses in truth rather than relying solely on compressed training data.

Why It Matters

The Problem

AI text prediction models tend to be more confident than accurate with Biblical texts. While they perform well on commonly cited passages (like John 3:16) in popular translations (NIV, KJV), accuracy degrades significantly with:

  • Lesser-quoted texts (minor prophets, Numbers, etc.)
  • Rare or minority language translations
  • Verses that lack extensive Internet training data

The Solution

By providing extensive contextual data for every verse broken into smaller task-focused files, AI systems can be grounded in truth. Since computers can read and hold an entire book in memory in under 1 second, we provide a book's worth of information for every verse.

Get Started

Installation

This project works with Claude Code, an AI agent that helps create and manage the commentary data.

# Install Claude Code
npm install -g @anthropic-ai/claude-code

# Clone the repository
git clone https://github.com/authenticwalk/context-grounded-bible.git
cd context-grounded-bible

# Start Claude Code
claude

Or use Claude Code directly at claude.ai/code

Using the Data

Simple Command Line Access

Get all commentary data for a specific verse:

cat ./bible/MAT/5/3/*.yaml > merged.yaml

Data Structure

All generated commentary follows this directory structure:

./bible/{book}/{chapter}/{verse}/{book}-{chapter}-{verse}-{task}.yaml

Example: ./bible/MAT/5/3/MAT-5-3-greek-words.yaml

Contributing

This project uses an AI agent self-learning loop to generate data:

  1. Human asks AI a question (e.g., "What are all the Greek words used in every verse?")
  2. AI clarifies goals and possibilities
  3. AI creates a sample data file and requests feedback
  4. Human and AI agents provide feedback
  5. AI refines its instructions through iteration
  6. Process the entire Bible with the optimized approach
